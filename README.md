### electron  + vite + Vue3 + TypeScript + SerialPort + pinia + Socket + ElementPlus + gRPC
* 前提条件
* node必须16.16.0以上版本。
* 不需要安装python，node-gyp等对应的编码解析工具
* 安装electron对应的版本
* 用的node-serialPort方式导入
* mac打包的时候 sudo npm run build 解决可能会出现打包空间不足的报错问题
* 目前采用node-gRPC方式引用
